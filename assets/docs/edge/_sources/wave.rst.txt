****************
Wave containers
****************

Wave is a container provisioning and augmentation service. It allows you to build containers for Nextflow pipeline
on-demand and allows the extension of *any* existing containers with pipeline provided scripts and other assets.


Getting started
===============

Wave can be used in any Nextflow pipeline adding the following plugin declaration in the `nextflow.config` file::

```
plugins {
  'nf-wave'
}
```
