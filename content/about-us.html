About Nextflow

Nextflow is free open source software distributed under the Apache 2.0 licence developed by Seqera Labs. The software is used by scientists and engineers to write, deploy and share data-intensive workflows on any infrastructure.

The project was started by Paolo Di Tommaso while at the Centre for Genomic Regulation (CRG).

Check it out the project repository on GitHub for details.
